Foundation Year Project

This unit is more aligned with the degree that you will follow given that it will allow you to become more fluent in reading and writing code as it's more relevant to your chosen area of study i.e. computer science. The unit is divided into 4 tasks which follow this order:

  1. Introduction to Linux
  2. Bash Scripting
  3. LaTeX
  4. HTML, CSS and Bootstrap
    5.

All of this must be mastered to a certain acceptable standard dictated by the coordinator in charge of overseeing student progress. Once all these tasks are complete the student is grouped with other students to complete a set project which is to create a website for prospective students interested in joining for a foundation year in computer science.

What is achieved from completion:

  • Analytical skills
  • Group/team working
  • Leadership
  • Project management
  • Oral communication
  • Problem solving
  • Research
  • Written communication
    •

These skills in the long term build a student's work ethic and forces the student to become more adaptable through individual and group tasks which helps increase the employability standard of that student.

The recommended study hours for this unit is:

  • Lectures - 2
  • Practical classes & workshops - 36
  • Independent study - 59
